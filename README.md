# wiredin.js


How to use:

1. Import the module
```js
import * as WiredIn from 'wiredin-node';
```
2. Log out the status
```js
console.log(WiredIn.status());
```
3. Change the color
```js
WiredIn.update.color('FF2DAA');
```
4. Change to a preset
```js
WiredIn.preset.name('Wired In');
```
